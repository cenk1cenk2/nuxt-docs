## Install Latest Version

Add `listr2` dependency to your project:

<code-group>
  <code-block label="Yarn" active>

```bash
yarn add listr2
```

  </code-block>
  <code-block label="NPM">

```bash
npm install listr2
```

  </code-block>
</code-group>
